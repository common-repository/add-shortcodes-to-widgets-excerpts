Add Shortcodes To Widgets And Excerpts allows you to add shortcodes to Wordpress Widget areas and Excerpts.

== Description ==

This free plugin permits you add shortcodes to Wordpress Widget and Excerpts

== Installation ==

* Upload the directory '/add-shortcodes-to-widgets-and-excerpts/' to your WP plugins directory and activate from the dashboard of the main blog.

* No further settings required
